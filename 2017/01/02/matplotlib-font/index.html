发表于 2017-01-02

Overview

Matplotlib这个库对比起D3.js, Google chart之类的前端可视化工具来说, 美观度和交互性都有所欠缺, 但是对于学术类的图片, 前两个显的太浮夸. 这时候就需要适当调整Matplotlib, 使得绘图不那么古板.

字体

matplotlib中最蛋疼的东西大概就是字体了，主要的问题在中文的支持问题，还有字体格式的问题。
